# Research Data

本目录用于存放仅供本地研究和内部核验使用的下载包、截图、查询结果和人工整理中间件。

建议结构：

- `tdt/`: 天地图相关研究材料
- `tencent/`: 腾讯位置服务相关研究材料
- `baidu/`: 百度地图开放平台相关研究材料
- `gaode/`: 高德开放平台相关研究材料
- `scu/`: 四川大学官方校区图、人工标注与核验记录

每个来源下建议继续拆分为：

- `raw/`: 原始下载包、接口响应、截图
- `notes/`: 获取时间、查询参数、校核说明
- `derived/`: 派生 overlay、人工修订结果

建议每批数据至少记录：

- 来源平台
- 官方入口 URL
- 获取日期
- 查询范围
- 查询关键词或接口参数
- 是否允许长期本地保存
- 是否允许导入 `normalized/` 或 `runtime/`

这些内容默认不纳入版本管理。
