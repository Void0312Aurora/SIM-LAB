# 城市转游戏资产与群体模拟项目想法

## 1. 项目愿景

我们希望构建一个项目，将现实世界中的城市、街区或局部建筑群转换为可交互的游戏/模拟场景，并在其上部署多类 NPC 与规则系统，观察复杂环境中群体行为的演化。

这个项目既可以朝“可玩的游戏原型”发展，也可以朝“偏研究和可视化的模拟平台”发展。两者的公共底座是：

- 现实空间数据的导入与清洗
- 可导航场景的自动生成
- 多阵营 NPC 的行为建模
- 事件驱动的模拟与结果回放

## 2. 核心吸引力

相比完全虚构地图，这个项目的独特性在于“真实空间 + 规则模拟”的结合：

- 玩家或观察者会天然对真实地点产生代入感
- 地图结构会对感染、逃生、封锁、冲突结果产生明显影响
- 同一个街区在不同规则、不同兵力和不同初始条件下，会出现完全不同的演化结果

这意味着项目既有“玩法可能性”，也有“实验价值”和“内容传播性”。

## 3. 场景与阵营设想

初期建议围绕少量高辨识度场景展开，例如：

- 密集居民街区
- 商业区/步行街
- 校园或园区
- 地铁站周边
- 工业区或仓储区

可先设计以下 NPC/阵营：

- 平民：目标是生存、逃离、寻找庇护或家人
- 感染者/丧尸：目标是扩散、追逐、受声音或视野刺激
- 军警/武装单位：目标是封锁、救援、清剿、维持秩序
- 医护/救援单位：目标是转移伤员、建立安全区

## 4. 玩法或模拟的核心问题

项目可以围绕这些问题来构建体验：

- 哪些街区最容易发生拥堵和踩踏？
- 在什么地形结构下，感染扩散最快？
- 何种封锁策略效果最好？
- 平民在恐慌状态下会如何改变路径选择？
- 如果军队、感染者和平民同时存在，会形成哪些非预期结果？

如果做成游戏，玩家可以扮演指挥者、幸存者或观察者。
如果做成模拟平台，重点则是参数设置、批量运行和结果可视化。

## 5. 推荐的第一阶段方向

不建议一开始做“整座城市 + 写实画面 + 大量单位”的大项目。

更建议的第一阶段是：

- 只选择一个真实街区
- 地图规模控制在 500m x 500m 到 1km x 1km
- 只做 3 类核心 NPC：平民、感染者、军队
- 先使用低模或简化风格资产
- 先做第三人称上帝视角、俯视角或 2.5D，而不是第一人称开放世界

这样可以把精力集中在真正决定项目成败的系统上：

- 地图导入是否稳定
- 导航与寻路是否可靠
- NPC 行为是否足够“像样”
- 模拟结果是否有解释性和趣味性

关于表现方向，当前更偏向于先把项目做成“宏观可观察、可控制、可回放”的上帝视角模拟/游戏原型。第一人称沉浸式表现可以作为后续阶段的增强方向，建立在前期资产流水线和模拟规则已经稳定的前提之上。

## 6. MVP 设想

一个合理的 MVP 可以包含：

- 导入一个真实街区的道路和建筑轮廓
- 自动生成简化场景体块
- 生成三类 NPC，并给出基础属性
- 支持视野、听觉、寻路、感染和恐慌规则
- 支持开始/暂停/倍速/回放
- 输出基础统计结果：
  - 存活人数
  - 感染人数
  - 疏散效率
  - 封锁持续时间
  - 高风险区域热力分布

## 6.1 当前首发地图选择

当前首发地图已经收敛为：

- 城市：成都
- 具体地点：四川大学江安校区
- 工程区域：江安校区核心区

选择这个地点的原因是：

- 校园场景边界相对清晰，适合首版模拟原型
- 道路、建筑、出入口和开放空间具有代表性
- 便于后续做现实结构和行为逻辑的人工校核

## 7. 关键技术模块

后续讨论技术栈时，可以按模块拆分：

- 地图数据来源与清洗
- 场景生成与资产流水线
- 导航网格与寻路
- NPC 行为系统
- 仿真调度与性能优化
- 可视化与调参工具
- 数据记录与回放系统

## 8. 主要风险

这个项目可行，但要注意以下风险：

- 真实地图数据质量不稳定，导入后可能需要大量清洗
- 如果太早追求写实，会被资产生产吞掉大部分时间
- 如果规则很多但缺乏重点，模拟可能会复杂但不好玩
- 单位数量一上来太大，会立刻遇到性能瓶颈
- 如果未来商业化，需要考虑地图数据授权与现实地点表达边界

## 9. 下一步讨论重点

下一轮建议集中讨论这些问题：

1. 项目更偏游戏，还是更偏模拟平台？
2. 首个演示地图用哪类现实街区？
3. 技术栈选 3D 引擎、Web 可视化，还是混合方案？
4. NPC 行为先做简单状态机，还是更复杂的 GOAP/行为树？
5. 地图资产优先走“自动生成”，还是“半自动人工修饰”？
6. 首发资产精度应该以“结构可信”还是“近距离写实”为目标？

## 10. 当前结论

项目是可行的，但最佳路线不是先做大而全的末日城市游戏，而是先做一个小范围、高解释性、可回放的真实街区模拟原型。只要这个原型成立，后续无论扩成游戏、研究工具还是内容平台，都会顺很多。

当前阶段已经进一步冻结：

- 首发地图：成都 · 四川大学江安校区核心区
- 首发形态：上帝视角真实空间模拟原型
